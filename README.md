# hello-vue-router
实现了对 vue-router 的手写实现，主要包括：
+ hash/history 模式路由
+ 嵌套路由
+ router-view / router-link
+ router / route
+ push / replace / go / back 等方法
+ router hook

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
